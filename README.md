Plugin: Converslab Notes
Desc: “Small content items created by admin, shown on frontend in different ways”

What it does (small but real):
Admin React screen to manage “Notes” (create/edit/delete/list)
Notes stored as Custom Post Type (converslab_note) + meta (e.g., priority, source_url)
REST API CRUD for notes
Dynamic Gutenberg block: “Converslab Notes” → shows latest notes with filters (priority, count)
